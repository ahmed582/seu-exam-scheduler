# SEU Exam Schedule PDF Generator

An installable browser application that converts Saudi Electronic University course exam CSV exports into polished, printable college schedules.

## Features

- Reads course CSV exports locally in the browser.
- Separates Bachelor and Master schedules.
- Supports midterm and final exam exports.
- Groups exams by college, day, and session.
- Prints all schedules, one college, or one individual schedule.
- Supports on-campus and online examination notes.
- Installs as a progressive web app.

## Privacy

Uploaded CSV files are processed inside the browser. The application does not upload or store the file contents.

## Development

```bash
npm install
npm run dev
```

## Publishing

Pushes to `main` are automatically built and deployed through GitHub Pages.
